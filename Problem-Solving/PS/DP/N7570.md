### Longest Increment Subsequence

```java
int[] dp = new int[N+1];

for(int i=1;i<=N;i++){
    dp[students[i]] = dp[students[i]-1]+1;
}

int max = 0;
for(int i=1;i<=N;i++){
    max = Math.max(max, dp[i]);
}
if(N == 1)System.out.println(0);
else System.out.println(N - max);
```
